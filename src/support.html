Questions fréquentes

Comment acheter un composant ?

Depuis la fiche d'un composant payant, appuyez sur le bouton d'achat. Le paiement est géré de manière sécurisée par Apple via votre identifiant Apple. Le composant est immédiatement accessible après l'achat.

Comment publier un composant ?

Connectez-vous, rendez-vous dans l'onglet "Publier", renseignez le nom, la description, le code SwiftUI et une image de prévisualisation. Choisissez un prix (gratuit ou payant) et soumettez.

Comment recevoir mes versements en tant que créateur ?

Renseignez votre adresse email PayPal dans votre profil. Les versements sont effectués automatiquement chaque mois pour toutes les ventes du mois précédent.

J'ai oublié mon mot de passe.

Sur l'écran de connexion, appuyez sur "Mot de passe oublié ?". Entrez votre email, vous recevrez un code à 6 chiffres valable 15 minutes pour réinitialiser votre mot de passe.

Comment supprimer mon compte ?

Envoyez une demande de suppression à contact@codaslibrary.app depuis l'adresse email de votre compte. Toutes vos données seront effacées sous 30 jours.

Un composant acheté n'apparaît pas dans mes obtenus.

Déconnectez-vous puis reconnectez-vous. Si le problème persiste, contactez-nous avec votre identifiant de transaction Apple.

Vous n'avez pas trouvé votre réponse ?

Notre équipe répond généralement dans les 24h.
